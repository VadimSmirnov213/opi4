\section{Суть задания}
В рамках лабораторной работы требовалось разработать сценарий сборки на Apache Ant
для проекта по дисциплине <<Веб-программирование>>.
Сценарий должен включать цели компиляции, сборки jar, запуска тестов, генерации документации,
конвертации локализаций native2ascii и дополнительные цели history/team/alt/music.

\section{Что было реализовано}
\begin{itemize}
    \item Добавлен сценарий \texttt{build.xml} со всеми требуемыми целями.
    \item Параметры и константы вынесены в \texttt{build.properties}.
    \item Реализованы служебные скрипты SVN-автоматизации в \texttt{tools/}.
    \item Добавлены JUnit-тесты для сервиса проверки попадания и валидаторов.
    \item Добавлены файлы локализации \texttt{i18n/messages*.properties} и их использование в коде.
    \item Настроено формирование \texttt{MANIFEST.MF} с версией, главным классом, датой сборки и checksum.
\end{itemize}
